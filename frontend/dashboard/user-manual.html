LoanPro – User Manual

Fintech Standard Guide

1. Introduction

Welcome to LoanPro, a digital platform designed to help users:

  • Discover loan schemes
  • Apply for loans easily
  • Track repayments
  • Manage financial activities securely

Built with a focus on transparency, speed, and user trust.

2. Account Registration & Login

Registration Methods
  • Traditional Email + Password
  • Google OAuth (quick signup)
Security Features
  • Email OTP Verification (ensures real users)
  • Encrypted password storage
  • Session-based authentication
Required Details
  • First Name & Last Name
  • Email Address
  • Password

After registration, users must verify their email before accessing features.

3. Dashboard Overview

The dashboard acts as your financial control center. It displays:

  • Active Loans
  • EMI Due Status
  • Loan Summary
  • Quick Navigation

Designed to give a quick snapshot of your financial position.

4. Apply for Loans

Features:
  • Browse available loan schemes
  • Filter by: Loan Type, Bank, Amount
Each Loan Card Shows:
  • Loan Name, Interest Rate, Tenure
  • Min/Max Amount, Processing Fee
Application Flow:
  • Select a loan scheme and click Apply
  • Submit required details
  • Wait for admin approval

5. My Loans Section

Displays:
  • All applied loans and their status (Pending, Approved, Rejected, Active)
  • Additional info: Loan amount, Interest rate, Remaining tenure

6. Payment & EMI Management

Features:
  • View EMI schedule, pay installments, and track payment status
Payment History:
  • Date of payment, Amount paid, Remaining balance

Ensures full financial transparency.

7. EMI Calculator

Helps users estimate monthly EMI by adjusting:

  • Loan amount, Interest rate, and Tenure

Useful before applying for a loan.

8. Profile Management

  • Update personal details
  • Change password
  • Manage account settings

9. Notifications

Alerts include:
  • Loan approval/rejection
  • EMI due reminders
  • Payment confirmations

10. Help & Support

Options:
  • Help Center
  • Contact Support

Designed to resolve issues quickly.

11. Security & Privacy

Measures:
  • Password hashing (bcrypt)
  • Secure API communication
  • Data validation

User data is handled with high confidentiality.

12. Important Guidelines

  • Always verify loan details before applying
  • Pay EMIs on time to avoid penalties
  • Keep login credentials secure

13. Future Enhancements

  • Credit score tracking
  • AI-based loan recommendations
  • Advanced analytics dashboard
